EncryptSecret cannot recover deleted notes. Once a note is read or its expiry time passes, it is permanently deleted from the database — by design. There is no way to get it back.

How to use EncryptSecret
01

Write your secret message

On the home page, type anything you want to share privately — a password, a personal note, sensitive info. Only the person you send the link to will ever see it.

02

Set your options

Choose how long the note lives (1 hour to 7 days), whether it burns after one read or allows multiple views, and optionally protect it with a password.

03

Copy and share the link

After creating the note you'll get a unique link. Send it to your recipient over WhatsApp, email, Telegram — any platform you like.

04

Recipient opens the link

When they open it, EncryptSecret decrypts the message directly in their browser. If burn-after-reading is on, the note is deleted from the database the moment they view it.

05

Note is gone forever

After the note is read or expires, it is permanently deleted. Anyone trying to open the same link again will find nothing — because nothing remains.
